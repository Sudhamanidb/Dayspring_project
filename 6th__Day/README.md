## Ruby Loops

* **for loop**: runs for a known range; auto-increments; inclusive range with `..`
* **while loop**: runs while condition is true; requires manual initialization and increment
* **loop do**: infinite loop; requires `break` to exit
* **until loop**: opposite of while; runs while condition is false

### Loop Control Keywords

* `break` â†’ exits the loop
* `next` â†’ skips current iteration
* `redo` â†’ repeats current iteration

## Enumerable Methods

* **select** â†’ returns values matching the condition (non-destructive)
* **select!** â†’ destructive version; modifies original array
* **reject** â†’ returns values not matching the condition (non-destructive)
* **reject!** â†’ destructive version; modifies original array
* **any?** â†’ returns true if at least one element meets the condition
* **all?** â†’ returns true if all elements meet the condition
* **map / collect** â†’ return a new array based on operation
* **each** â†’ returns original array; useful for iteration
